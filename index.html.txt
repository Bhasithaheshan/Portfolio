Ultra-Premium Portfolio

Create an ultra-premium, futuristic single-page portfolio website for **Bhasitha Heshan**, designed to feel like a high-end SaaS product or Apple-level landing page.

### 🧠 Core Goal:

The site must feel like a **modern web app**, not a basic portfolio. Prioritize smoothness, micro-interactions, and premium UI/UX.

---

## ⚙️ Tech Stack:

* HTML5
* Tailwind CSS (via CDN)
* GSAP (core + ScrollTrigger + ScrollSmoother if possible)
* Lucide Icons CDN
* Vanilla JavaScript only (no frameworks)

---

## 🎨 Design System:

* Theme: Dark futuristic (black / deep gray / neon accents)
* Accent Colors: Electric blue (#00f0ff), purple glow (#8b5cf6)
* Typography: Use modern fonts (e.g., Inter / Poppins via Google Fonts)
* Glassmorphism + soft gradients
* Subtle grid/mesh background

---

## 🚀 Advanced Features & Animations:

### 1. Hero Section (Cinematic Entry)

* Fullscreen hero with animated gradient or particle background
* Name: **Bhasitha Heshan**
* GSAP stagger animation:

  * Each letter reveals from bottom with blur + opacity transition
* Add a typing animation for roles:

  * "Full-stack Developer"
  * "Algorithmic Trader"
  * "Pine Script Expert"
* Include a subtle floating/parallax effect

---

### 2. Smooth Scrolling System

* Implement GSAP ScrollSmoother (or custom smooth scroll fallback)
* Add inertia/momentum scrolling
* Sections must feel fluid like macOS scrolling

---

### 3. Scroll-Based Animations

* Use GSAP ScrollTrigger:

  * Sections fade in + slide up
  * Elements stagger into view
  * Add slight scale animation (0.95 → 1)
* Add parallax scrolling for background elements

---

### 4. About Section

* Clean glass card with short intro
* Animated counters (e.g., Projects, Experience, Win Rate)
* Icons using Lucide

---

### 5. Project Showcase (Advanced Grid)

Display 3 projects:

1. Liquidity Sweep Pro (Screenshot (819).png)

3. Sniper Entry V.02 (Screenshot (818).png)

* Overlay appears with:

---

### 6. Interactive Effects

* Cursor glow or custom cursor (optional)
* Hover micro-interactions everywhere
* Buttons with animated gradient borders

---

### 7. Background Effects

Choose ONE:

* Animated particle background (canvas)
  OR
* Moving gradient mesh background

Must be subtle, smooth, and not laggy.

---

### 8. Contact / Footer Section

* Futuristic contact area with glowing buttons:

  * GitHub → https://github.com/Bhasithaheshan 
  * WhatsApp → https://wa.me/94703187763 
  * Email → mailto:bhasithaheshan@gmail.com
* Add hover glow + scale animation

## ⚡ Performance Requirements:

* Fully responsive (mobile + desktop)
* Optimize animations for performance
* Use will-change and transform wisely
* Avoid heavy libraries

## 📦 Output Format:

* Provide a **single complete HTML file**

        // ===== LOADING SCREEN =====
        const loader = document.getElementById('loader');
        const loaderProgress = document.getElementById('loader-progress');
        const loaderPercent = document.getElementById('loader-percent');
        let loadProgress = 0;

        function updateLoader() {
            loadProgress += Math.random() * 15 + 5;
            if (loadProgress > 100) loadProgress = 100;
            loaderProgress.style.width = loadProgress + '%';
            loaderPercent.textContent = Math.floor(loadProgress) + '%';

            if (loadProgress < 100) {

        // ===== PARTICLE BACKGROUND =====
        const canvas = document.getElementById('particle-canvas');
        const ctx = canvas.getContext('2d');
        let particles = [];
        const particleCount = 60;

        function resizeCanvas() {
            canvas.width = window.innerWidth;
            canvas.height = window.innerHeight;
        }
        resizeCanvas();
        window.addEventListener('resize', resizeCanvas);

        class Particle {
            constructor() {
                this.reset();
            }

            reset() {
                this.x = Math.random() * canvas.width;
                this.y = Math.random() * canvas.height;
                this.size = Math.random() * 2 + 0.5;
                this.speedX = (Math.random() - 0.5) * 0.5;
                this.speedY = (Math.random() - 0.5) * 0.5;
                this.opacity = Math.random() * 0.5 + 0.1;
                this.color = Math.random() > 0.5 ? '#00f0ff' : '#8b5cf6';
            }

            update() {
                this.x += this.speedX;
                this.y += this.speedY;

                if (this.x < 0 || this.x > canvas.width || this.y < 0 || this.y > canvas.height) {
                    this.reset();
                }
            }

            draw() {
                ctx.beginPath();
                ctx.arc(this.x, this.y, this.size, 0, Math.PI * 2);
                ctx.fillStyle = this.color;
                ctx.globalAlpha = this.opacity;
                ctx.fill();
                ctx.globalAlpha = 1;
            }
        }

        function initParticles() {
            particles = [];
            for (let i = 0; i < particleCount; i++) {
                particles.push(new Particle());
            }
        }
        initParticles();

        function animateParticles() {
            ctx.clearRect(0, 0, canvas.width, canvas.height);

            particles.forEach(p => {
                p.update();
                p.draw();
            });

            // Draw connections
            particles.forEach((p1, i) => {
                particles.slice(i + 1).forEach(p2 => {
                    const dx = p1.x - p2.x;
                    const dy = p1.y - p2.y;
                    const dist = Math.sqrt(dx * dx + dy * dy);

                    if (dist < 150) {
                        ctx.beginPath();
                        ctx.moveTo(p1.x, p1.y);
                        ctx.lineTo(p2.x, p2.y);
                        ctx.strokeStyle = '#00f0ff';
                        ctx.globalAlpha = (1 - dist / 150) * 0.1;
                        ctx.lineWidth = 0.5;
                        ctx.stroke();
                        ctx.globalAlpha = 1;
                    }
                });
            });

            requestAnimationFrame(animateParticles);
        }
        animateParticles();

                card.querySelector('.project-card-inner').style.transform = 
                    `perspective(1000px) rotateX(${rotateX}deg) rotateY(${rotateY}deg) scale(1.02)`;
            });

            card.addEventListener('mouseleave', () => {
                card.querySelector('.project-card-inner').style.transform = 
                    'perspective(1000px) rotateX(0) rotateY(0) scale(1)';
            });
        });

        // ===== MOBILE MENU =====
        const mobileMenuBtn = document.getElementById('mobile-menu-btn');
        const mobileMenu = document.getElementById('mobile-menu');

        mobileMenuBtn.addEventListener('click', () => {
            mobileMenu.classList.toggle('hidden');
        });

        // Close mobile menu on link click
        mobileMenu.querySelectorAll('a').forEach(link => {
            link.addEventListener('click', () => {
                mobileMenu.classList.add('hidden');
            });
        });

        // ===== SMOOTH SCROLL =====
        document.querySelectorAll('a[href^="#"]').forEach(anchor => {
            anchor.addEventListener('click', function(e) {
                e.preventDefault();
                const target = document.querySelector(this.getAttribute('href'));
                if (target) {
                    target.scrollIntoView({
                        behavior: 'smooth',
                        block: 'start'
                    });
                }
            });
        });

        // ===== NAV BACKGROUND ON SCROLL =====
        const nav = document.querySelector('nav');
        window.addEventListener('scroll', () => {
            if (window.scrollY > 50) {
                nav.classList.add('nav-glass');
            } else {
                nav.classList.remove('nav-glass');
            }
        });
    </script>
